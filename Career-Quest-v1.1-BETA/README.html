Career Quest README

Thank you for checking out Career Quest!

Career Quest is an interactive career exploration game where you experience different professions through short, skill-based challenges. Your goal is to explore career paths, understand what each role involves, and build a better sense of the skills used in those jobs.

How to Run

Currently, we're in the developer beta, so setup might be a bit more complex, but future versions will be easier to run!

  1. Make sure Python is installed
    You can download Python from here.
  2. Install Pygame
    Pygame is the framework we use to build the game. Install it by running the following command in your terminal: pip install pygame
  3. Run the Game
    To start the game, open the terminal in the folder containing main.py and run: python main.py

Controls

  • Enter – Start the game and advance tutorial dialogue
  • Mouse – Click buttons, select career paths, and interact with minigames
  • Esc – Quit the game or leave a career world

Features

  • Explore different career worlds and see what those jobs can feel like
  • Playable Software Developer and Chef experiences
  • Skill-based minigames focused on problem solving, timing, precision, and logic
  • JSON-based save system to track completed careers and skill progress
  • Interactive UI with buttons, dialogue, and responsive layouts
  • Coming soon careers such as Pilot and Detective
